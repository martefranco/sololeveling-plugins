# Herramientas: Plataformas Digitales

**Alcance:** Transversal — marketplaces, e-commerce, y plataformas digitales

## Marketplaces

### Amazon (Seller Central)
- **Categoría:** Marketplace / e-commerce
- **Uso típico:** Gestión de inventario, listings, publicidad, fulfillment (FBA), atención a vendedores
- **Roles que la usan:** E-commerce managers, soporte a vendedores, logística, marketing digital
- **Nivel de dominio esperado:** Variable según rol

### Mercado Libre (Seller Center)
- **Categoría:** Marketplace / e-commerce LATAM
- **Uso típico:** Publicaciones, métricas de reputación, Mercado Envíos, atención a compradores
- **Roles que la usan:** E-commerce, soporte, logística, customer experience
- **Nivel de dominio esperado:** Variable según rol

### Shopify
- **Categoría:** Plataforma de e-commerce
- **Uso típico:** Tiendas en línea, gestión de productos, pagos, envíos
- **Roles que la usan:** E-commerce, marketing digital, diseño web
- **Nivel de dominio esperado:** Básico a Intermedio

## Plataformas de comunicación

### Slack
- **Categoría:** Comunicación interna
- **Uso típico:** Mensajería de equipo, canales, integraciones, workflows
- **Roles que la usan:** Equipos tech, operaciones, gestión de proyectos
- **Nivel de dominio esperado:** Básico

### Microsoft Teams
- **Categoría:** Comunicación y colaboración
- **Uso típico:** Chat, videollamadas, archivos compartidos, integración con Office 365
- **Roles que la usan:** Todos los roles en empresas que usan Microsoft
- **Nivel de dominio esperado:** Básico

### Zoom
- **Categoría:** Videoconferencias
- **Uso típico:** Reuniones virtuales, webinars, grabaciones
- **Roles que la usan:** Todos
- **Nivel de dominio esperado:** Básico
