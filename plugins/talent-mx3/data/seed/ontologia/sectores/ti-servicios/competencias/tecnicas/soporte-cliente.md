# Competencias Técnicas: Soporte al Cliente

**Sector:** TI / Servicios Externalizados
**Área:** Atención al Cliente

## Competencias

### Gestión de tickets
- **Definición:** Capacidad de crear, clasificar, priorizar, escalar y resolver tickets en un sistema de helpdesk siguiendo SLAs definidos
- **Herramientas asociadas:** Zendesk, Freshdesk, ServiceNow, Jira Service Management
- **Cómo se evalúa:** Ejercicio práctico — dar un ticket ejemplo y pedir que lo clasifique, priorice y redacte respuesta
- **Niveles:** Básico: crear y clasificar / Intermedio: escalar y resolver N1 / Avanzado: análisis de tendencias y mejora de procesos

### Atención multicanal
- **Definición:** Capacidad de atender clientes por múltiples canales (chat, correo, teléfono, redes sociales) adaptando tono y formato
- **Herramientas asociadas:** Zendesk (chat + correo), Genesys (teléfono), Hootsuite (redes)
- **Cómo se evalúa:** Role play — simular atención por chat vs. teléfono, verificar adaptación de tono
- **Niveles:** Básico: un canal / Intermedio: 2-3 canales / Avanzado: omnicanal con contexto cruzado

### Diagnóstico de problemas
- **Definición:** Capacidad de identificar causa raíz de un problema reportado por el cliente usando lógica de descarte y árboles de decisión
- **Herramientas asociadas:** Base de conocimiento, herramientas de diagnóstico del cliente, logs
- **Cómo se evalúa:** Caso práctico — presentar un escenario de problema y pedir que identifique causa raíz
- **Niveles:** Básico: sigue script / Intermedio: diagnóstica sin script / Avanzado: crea scripts para otros

### Documentación de casos
- **Definición:** Capacidad de registrar cada interacción con detalle suficiente para que otro agente pueda continuar sin pedir al cliente que repita información
- **Herramientas asociadas:** Sistema de ticketing, Confluence, Notion
- **Cómo se evalúa:** Revisar ejemplo de documentación — ¿tiene contexto, acciones tomadas, resultado?
- **Niveles:** Básico: registra / Intermedio: registra con estructura / Avanzado: crea templates de documentación

### Métricas de servicio
- **Definición:** Comprensión de KPIs de soporte: CSAT, NPS, FCR, AHT, SLA compliance, y capacidad de autogestionarse para cumplirlos
- **Herramientas asociadas:** Dashboards de Zendesk, Power BI, reportes del CRM
- **Cómo se evalúa:** Preguntar qué métricas conoce, cómo las usaba, qué hacía si no cumplía
- **Niveles:** Básico: conoce las métricas / Intermedio: se autogestiona para cumplirlas / Avanzado: propone mejoras basadas en datos

### Redacción profesional
- **Definición:** Capacidad de comunicar por escrito de forma clara, empática, sin errores ortográficos, adaptando el nivel de tecnicidad al interlocutor
- **Herramientas asociadas:** Correo, chat, base de conocimiento
- **Cómo se evalúa:** Ejercicio de redacción — pedir que escriba respuesta a un cliente molesto
- **Niveles:** Básico: sin errores / Intermedio: empático y claro / Avanzado: persuasivo y profesional
